# 📌 Express Post Manager (CRUD App)

A full-stack **CRUD (Create, Read, Update, Delete)** application built using **Node.js, Express, MySQL, and EJS**.
This project allows users to create, view, edit, and delete posts with a clean and responsive UI.

---

## 🚀 Features

* ✍️ Create new posts
* 📖 View all posts
* ✏️ Edit existing posts
* 🗑️ Delete posts with confirmation
* 🎨 Stylish UI with custom CSS
* ⚡ Fast backend using Express & MySQL

---

## 🛠️ Tech Stack

* **Frontend:** HTML, CSS, EJS
* **Backend:** Node.js, Express.js
* **Database:** MySQL
* **Other Tools:** Method-Override, UUID

## ⚙️ Installation & Setup

1. Clone the repository

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Navigate to project folder

```
cd your-repo-name
```

3. Install dependencies

```
npm install
```

4. Setup MySQL database

* Create a database
* Create a table `post` with fields:

```
id (VARCHAR / PRIMARY KEY)
name (VARCHAR)
comment (TEXT)
```

5. Start the server

```
node index.js
```

6. Open in browser

```
http://localhost:8080/view
```
